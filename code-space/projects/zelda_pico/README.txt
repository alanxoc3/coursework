This is from a challenge to create a cool platform-fighter pico-8 game, using a
template. The group created a pathetic little Legend of Zelda themed game with
a Goomba and Pikachus in only an hour.

Open the html file in a browser to play it!

```
fighter.p8 - Original source file
zelda.p8   - Zelda source file
zelda.zip  - Contains the .js and .html files
```
